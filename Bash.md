# ✅ **BASH COMMANDS LIST (FULLY ORGANIZED)**

## **1. File & Directory Commands**

* `ls` – list directory contents
* `ls -a` – list all (including hidden)
* `ls -l` – long format
* `cd` – change directory
* `pwd` – print working directory
* `mkdir` – create directory
* `rmdir` – remove empty directory
* `rm` – remove file
* `rm -r` – remove folder recursively
* `cp` – copy files
* `cp -r` – copy directories
* `mv` – move/rename files
* `touch` – create empty file
* `cat` – view file contents
* `nano` – edit file (terminal editor)
* `vim` – edit file (advanced editor)
* `more` – paginated file reader
* `less` – better file reader
* `head` – first lines
* `tail` – last lines
* `tail -f` – live log output

---

## **2. File Permissions / Ownership**

* `chmod` – change permissions
* `chmod 755 file` – read/write/execute
* `chown` – change file owner
* `chgrp` – change group

---

## **3. Search Commands**

* `find` – search files in directory
* `locate` – find files via index
* `grep` – search text inside files
* `grep -r` – recursive search
* `grep -i` – ignore case
* `which` – shows command path
* `whereis` – find binary/source/manpage

---

## **4. System Information**

* `uname` – system info
* `uname -a` – all details
* `hostname` – system hostname
* `uptime` – system running time
* `top` – real-time processes
* `htop` – advanced top (sometimes needs install)
* `ps` – list running processes
* `ps aux` – show all processes
* `free -h` – memory usage
* `df -h` – disk usage
* `du -h` – directory space usage

---

## **5. Networking Commands**

* `ping` – test network connection
* `ifconfig` – show IP interface (older)
* `ip a` – show IP addresses (newer)
* `wget` – download files
* `curl` – send requests / download
* `ssh` – connect to remote server
* `scp` – copy files over SSH
* `netstat` – network stats (deprecated but used)
* `ss` – modern netstat replacement

---

## **6. Package Management**

### **Ubuntu / Debian (apt)**

* `sudo apt update`
* `sudo apt upgrade`
* `sudo apt install <package>`
* `sudo apt remove <package>`

### **RedHat / CentOS (yum / dnf)**

* `sudo yum install`
* `sudo yum update`

---

## **7. Compression / Archiving**

* `tar` – archive utility

  * `tar -cvf` – create archive
  * `tar -xvf` – extract
* `zip` – create zip file
* `unzip` – extract zip

---

## **8. User Management**

* `whoami` – current user
* `who` – logged-in users
* `id` – user ID
* `adduser` – add user
* `deluser` – delete user
* `passwd` – change password

---

## **9. Process Control**

* `kill` – kill process
* `kill -9` – force kill
* `killall` – kill by name
* `jobs` – list background jobs
* `bg` – send process to background
* `fg` – bring to foreground

---

## **10. Disk Management**

* `mount`
* `umount`
* `fdisk` – partition tool
* `blkid` – block device info

---

## **11. Bash Scripting Commands**

* `echo` – print message
* `read` – take input
* `source` – run file in same shell
* `bash script.sh` – run script
* `#!/bin/bash` – shebang

---

## **12. Text Processing Commands**

* `awk` – powerful text processing
* `sed` – text replace/edit
* `cut` – cut columns
* `sort` – sort lines
* `uniq` – remove duplicates
* `tr` – translate characters
* `wc` – word count

---

## **13. Navigation Shortcuts**

* `.` – current directory
* `..` – parent directory
* `~` – home directory
* `-` – previous directory

---

## **14. Environment & Variables**

* `export` – set environment variable
* `env` – list environment
* `printenv`
* `set` – list shell variables

---

## **15. System Control**

* `shutdown`
* `reboot`
* `sudo` – run as admin
* `systemctl start service`
* `systemctl enable service`
