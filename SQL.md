
# ✅ **1. DDL — Data Definition Language**

Used to define and modify database structure.

* **CREATE** – create database objects
* **ALTER** – modify table structure
* **DROP** – delete table or database
* **TRUNCATE** – delete all rows (cannot rollback)
* **RENAME** – rename a table
* **COMMENT** – add comments to metadata

---

# ✅ **2. DML — Data Manipulation Language**

Used to insert, update, delete data.

* **INSERT**
* **UPDATE**
* **DELETE**
* **MERGE** (used for UPSERT: insert or update)

---

# ✅ **3. DQL — Data Query Language**

Used to fetch data.

* **SELECT**

---

# ✅ **4. DCL — Data Control Language**

Used to control permissions.

* **GRANT**
* **REVOKE**

---

# ✅ **5. TCL — Transaction Control Language**

Used to manage transactions.

* **COMMIT**
* **ROLLBACK**
* **SAVEPOINT**
* **SET TRANSACTION**

---

# ✅ **6. SQL Clauses**

These are part of SELECT statement.

* **WHERE**
* **GROUP BY**
* **HAVING**
* **ORDER BY**
* **LIMIT / OFFSET**
* **DISTINCT**
* **BETWEEN**
* **LIKE**
* **IN**
* **IS NULL / IS NOT NULL**

---

# ✅ **7. SQL Operators**

Used inside conditions.

* **Arithmetic Operators** (+, –, *, /, %)
* **Comparison Operators** (=, <>, >, <, >=, <=)
* **Logical Operators** (AND, OR, NOT)
* **ANY, ALL**
* **EXISTS**

---

# ✅ **8. SQL Joins**

Used to combine rows from multiple tables.

* **INNER JOIN**
* **LEFT JOIN**
* **RIGHT JOIN**
* **FULL OUTER JOIN**
* **CROSS JOIN**
* **SELF JOIN**

---

# ✅ **9. SQL Constraints**

Applied on tables/columns to maintain data integrity.

* **PRIMARY KEY**
* **FOREIGN KEY**
* **UNIQUE**
* **NOT NULL**
* **CHECK**
* **DEFAULT**
* **INDEX**
